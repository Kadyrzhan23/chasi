# CHASI.UZ — Полная дизайн-спецификация

Люксовый бутик наручных часов (Ташкент). Тёмная премиум-эстетика. React SPA, витрина + личный кабинет + CRM. Две переключаемые дизайн-темы с одинаковым функционалом и данными.

---

## 1. Общая концепция

- **Настроение:** дорого, сдержанно, «тёмный люкс». Много воздуха, тонкие линии, золото/акцент на чёрном.
- **Язык интерфейса:** русский.
- **Две темы**, переключаются кнопкой в шапке (1 / 2):
  - **V1 · Noir** — чёрно-золотая классика, засечная типографика, SVG-графика часов, острые углы (radius 0).
  - **V2 · Onyx** — чистый чёрный + серебро и алый, гротеск, фото товаров, скруглённые углы (radius 22px, кнопки-пилюли).
- Ключевая механика витрины: **цены скрыты (блюр) для неавторизованных**, открываются после входа. Корзина и покупка — только для авторизованных.

## 3. Типографика

Шрифты подключены с Google Fonts:

- **V1 Noir:** засечный `Cormorant Garamond` (заголовки, цены, крупные цифры) + гротеск `Manrope` (текст, интерфейс).
- **V2 Onyx:** `Syne` (заголовки) + `Space Grotesk` (текст).
- Дополнительно загружены `Prata`, `Jost` (резерв).

| Элемент | Шрифт | Размер | Начертание | Трекинг |
|---|---|---|---|---|
| H1 `.big` | serif | `clamp(2.8rem, 5.6vw, 5.2rem)` | 500, `em` — курсив/акцентный цвет | — |
| H2 | serif | `clamp(1.9rem, 3.4vw, 3rem)` | 500 | — |
| H3 | serif | ~1.22rem | 500 | — |
| `.sec-label` (надзаголовок) | sans | `.72rem` | UPPERCASE, цвет `--gold` | `.32em` |
| Body | sans | ~.85–.9rem | 300 (тонкий) | — |
| Кнопки | sans | `.78rem` | UPPERCASE, 600 | `.16em` |
| Таблицы (шапка) | sans | `.66rem` | UPPERCASE, `--gold` | `.2em` |
| KPI-цифра | serif | `2.2rem`, цвет `--gold2` | — | — |

Базовые правила текста: тонкий вес (300) для абзацев, `line-height` ~1.7–1.8, обильные UPPERCASE-надписи с большим трекингом как акцент премиума.

## 4. Сетка, отступы, адаптив

- **Горизонтальные поля страниц:** `4vw`.
- **Шапка:** фиксированная сверху, высота ~76px, `backdrop-filter: blur(14px)`, полупрозрачный фон `--hdrbg`, нижняя граница `--line`.
- **Основные грид-раскладки:**
  - `.grid4` — 4 колонки (избранное на главной), gap 24px.
  - `.grid3` — 3 колонки (каталог, наборы), gap 24px.
  - Каталог: `290px | 1fr` (фильтры-сайдбар + сетка).
  - Товар: `1fr | 1.05fr` (медиа sticky + инфо).
  - Корзина: `1.5fr | 1fr` (товары + сводка sticky).
  - CRM: `240px | 1fr` (сайдбар-вкладки + контент).
- **Брейкпоинты:**
  - `≤1000px` — навигация прячется в бургер-меню; все двухколоночные раскладки → одна колонка; grid4/grid3 → 2 колонки; сайдбар CRM → горизонтальные табы.
  - `≤760px` — паспорт/спец-грид в 1–2 колонки.
  - `≤560px` — всё в одну колонку; уменьшенные заголовки/таблицы.

## 5. Компоненты

### Шапка (Header)
Логотип `CHASI.UZ` (serif, трекинг `.3em`, `.UZ` приглушён) слева · навигация по центру (Главная / Каталог / Наборы / Кабинет / CRM · демо) · справа: переключатель тем (два круглых пилюля-кнопки 1/2), иконка корзины со счётчиком, кнопка «Войти». Активный пункт навигации подчёркнут золотом.

### Кнопки
- `.btn-gold` — золотой градиент `135deg, --gold → --gold3`, текст `--ongold`, при наведении подъём `-2px` + мягкое золотое свечение.
- `.btn-ghost` — прозрачная, контур `--soft`, при наведении контур золотой.
- `.btn-sm` — уменьшенный вариант.
- Padding 15×34px, UPPERCASE, трекинг `.16em`, радиус `--btnrad` (0 в Noir / пилюля в Onyx).

### Карточка товара `.card`
Фон `--panel`, прозрачный контур, padding `30×24×26`. При наведении: подъём `-8px`, золотой контур, тень `--shadow`, «блик» пробегает по карточке (skew-градиент), иконка часов слегка увеличивается и поворачивается. Содержимое: визуал часов → название (serif) → бренд·диаметр → цена (блюр если не авторизован) → «Открыть карточку →».

**Угловые бейджи** (`.card-corner` left/right, стопкой):
- Слева сверху: тип товара — `Оригинал` (зелёный) / `1:1 клон` (синий), плюс золотой бейдж скидки `−N%`.
- Справа сверху: наличие — `в наличии` (зелёный) / `под заказ` (красный) / `осталось N` (красный, если ≤3).

### Чипы `.chip`
Пилюля с контуром `--line2`, при наведении золотой контур; активный (`.on`) — золотая заливка, текст `--ongold`. Используются в фильтрах, выборе скидки, статусов, оплаты.

### Теги / Пилюли статусов
- `.tag.orig` (зелёный) / `.tag.copy` (синий) — класс товара.
- `.pill.g/r/b/y` — цветные капсулы (зелёный/красный/синий/золото) для статусов в таблицах и карточках.

### Таблицы `.tbl`
Шапка: мелкий UPPERCASE золотой текст, нижняя золотая линия. Строки: тонкий вес, разделители `--line2`, при наведении лёгкая золотая подсветка строки.

### KPI-карточки `.kpi`
Контур `--line`, лёгкий золотой градиент-заливка сверху. Крупная serif-цифра `--gold2` + мелкая UPPERCASE-подпись + дельта (зелёная/красная).

### Панель `.panel` и бар-чарты
Контур `--line2`, фон `--panel`, padding 26px. Горизонтальные бары: дорожка `--wash`, заполнение — золотой градиент с анимацией ширины (1.2s ease-out) и значением справа.

### Тост-уведомления `.toast`
Всплывает снизу справа, ширина до 380px, выезд снизу (cubic-bezier). Два стиля:
- Telegram-стиль (тёмно-синий `#17212b`, синий заголовок) — имитация сообщения бота.
- `.gold` — фирменный (фон `--panel`, золотой заголовок).
Структура: шапка канала (иконка + подпись) + заголовок + текст.

### Формы
Инпуты `.search-inp` / `.select`: фон `--bg`, контур `--line2`, при фокусе золотой контур. Радио оплаты `.pay-opt` — строки-карточки, активная с золотым контуром и лёгкой заливкой. Степпер количества `.stepper` — кнопки −/N/+ с квадратно-круглым контуром.

### Страница товара
Слева — sticky медиа-блок (радиальное золотое свечение, визуал часов, теги статуса/скидки/остатка). Справа — надзаголовок бренда, крупный serif-заголовок, таблица из 8–11 характеристик, блок цены (зачёркнутая старая + новая + «выгода N» при скидке), баннер срочности `🔥 Осталось N / Последний экземпляр`, кнопки действий, примечание о доставке.

### Корзина
Слева — строки товаров (мини-визуал, название, кол-во-степпер, цена со скидкой), опция подарочного бокса (чекбокс), выбранный подарочный набор. Справа sticky — контакты/доставка, выбор оплаты (Payme / Click / при получении), сводка сумм и кнопка «Оформить заказ». Экран успеха с номером заказа.

### CRM (Admin)
Левый сайдбар-вкладки (Дашборд, Продукты·склад, Заказы с сайта, Новый заказ, Продажи, Интересы, Спрос, Trade-in, Скоро ТО, Клиенты). Активная вкладка — золотая левая полоса + подсветка. Контент: KPI, панели с барами, таблицы, карточки заказов, панель управления товарами (строки с контролами статус/остаток/скидка).

## 6. Эффекты и анимации

- **Reveal при скролле:** секции появляются снизу вверх (opacity + translateY 40px, 0.9s cubic-bezier).
- **Плавающие часы (hero):** вертикальное покачивание `floatY` 7s + лёгкий наклон, drop-shadow.
- **Бегущая строка брендов (marquee):** бесконечная горизонтальная прокрутка 28s.
- **Счётчики цифр:** анимированный отсчёт при попадании в вьюпорт (cubic ease-out ~1.8s).
- **Блик по карточке** при наведении; **бары** заполняются с анимацией ширины; **тосты** выезжают снизу.
- Общий easing проекта: `cubic-bezier(.16,1,.3,1)`.

## 7. Иконография и графика

- Логотип текстовый: `CHASI` + `.UZ` (приглушённый).
- Часы отрисованы **inline SVG** (компонент WatchSVG) — циферблат из двух цветов `dial` + акцент `accent`, опционально «живая» секундная стрелка. В теме Onyx вместо SVG — фото (Unsplash) с откатом на SVG при ошибке загрузки.
- Иконка корзины — тонкая SVG-сумка (stroke = текущий цвет, наследует золото/серебро), со счётчиком-бейджем.
- Декоративные символы-«искры»: `✦`, `✈` (в тостах), `🔥` (срочность), `🔒` (скрытая цена).

## 9. Краткая формула стиля (для брифа в другой сервис)

> Тёмный люкс-магазин часов. Фон почти чёрный (`#0a0a0d`), акцент — золото (`#d4af6a` / светлое `#f0d9a8`). Засечные заголовки Cormorant Garamond + тонкий гротеск Manrope. Много воздуха, тонкие полупрозрачные золотые линии, UPPERCASE-надписи с широким трекингом. Карточки на тёмных панелях с золотым контуром и «бликом» при наведении. Цены скрыты блюром до авторизации. Статусы — цветные капсулы (зелёный «в наличии/оригинал», красный «под заказ/мало», синий «копия»). Плавные появления по скроллу, easing `cubic-bezier(.16,1,.3,1)`. Альтернативная тема — чистый чёрный с алым акцентом (`#e5484d`), гротеск Syne/Space Grotesk, скруглённые углы и кнопки-пилюли.
