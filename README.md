# CHASI.UZ — демо-прототип для презентации клиенту

React + TypeScript + Vite. Все данные — мок, проект чисто демонстрационный.

## Запуск

```bash
npm install
npm run dev        # откроется на http://localhost:5173
```

Либо без запуска: откройте готовый файл `dist/index.html` двойным кликом (проект уже собран).

## Деплой

**Быстрый способ:** готовый архив `chasi-deploy.zip` лежит в корне проекта — это чистая продакшен-сборка (2 дизайн-версии: Noir и Onyx). Перетащите его на https://app.netlify.com/drop — публичная ссылка будет через минуту.

Приложение полностью статическое (HashRouter, относительные пути) — папку `dist/` можно выложить на любой хостинг без настройки сервера:

- **Netlify** (проще всего): зайдите на https://app.netlify.com/drop и перетащите папку `dist` — сайт получит публичный URL за минуту.
- **Vercel**: `npm i -g vercel && vercel --prod` в корне проекта (Framework: Vite, Output: dist).
- **GitHub Pages**: залейте содержимое `dist/` в ветку `gh-pages`.
- **Любой хостинг/VPS**: скопируйте содержимое `dist/` в корень веб-сервера.

Перед деплоем после правок: `npm run build`.

## Что показывать клиенту (сценарий демо)

1. **Главная** — премиальный дизайн, живые часы с реальным временем, анимации.
2. Кнопка **«Войти»** в шапке — цены скрыты до авторизации, после входа открываются (сбор базы клиентов).
3. В каталоге или на главной **кликните карточку товара 3 раза** — придёт «Telegram-уведомление» с персональной скидкой −10%.
4. **Каталог** — фильтры: класс товара (оригиналы / 1:1 клоны / AAAA / AAA), бренды, подбор по обхвату запястья ✦, «запас хода на выходные», водозащита для плавания, цена, стиль.
5. **Кабинет** — цифровые паспорта купленных часов (серийник, QR, гарантия, история ТО), wishlist с очередью, trade-in с загрузкой фото, уведомления. Кнопка «Демо: уведомление о ТО».
6. **CRM · демо** — панель владельца: аналитика просмотров за 7 дней, интересы клиентов (кнопка «Отправить −10%»), лист ожидания с депозитами и потенциальной выручкой, trade-in заявки, клиенты со скорым ТО (кнопка «Отправить напоминание» показывает, что увидит клиент), база клиентов с LTV.

Папка `chasi-redesign/` — первый статический прототип главной (можно не показывать).
